# AI Personal Assistant Chatbot

## Project Overview
AI Personal Assistant Chatbot is an AI-powered application developed using Python, Streamlit, and Groq API. 
The application provides a modern AI chat experience with intelligent conversation handling, memory-based interaction,
translation support, and real-time information responses through a single AI engine workflow.

The chatbot is designed with a clean and user-friendly interface inspired by modern AI assistant platforms.


## Features
- Intelligent AI conversation
- Memory-based chat interaction
- Translation support through natural prompts
- Real-time information and news handling
- Modern chatbot user interface
- Interactive AI response system
- User-friendly AI assistant workflow



## Technologies Used
- Python
- Streamlit
- Groq API
- News API
- Generative AI
